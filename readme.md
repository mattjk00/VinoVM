# VinoVM

The virtual machine for running vino language bytecode.

## Bytecode Example

```asm
0x1200000000000000 // Load Address 0
0xF100000000000000 // Debug Print String
0xFF00000000000000 // Terminate Program
0x48 // 'H'
0x69 // 'i'
0x0
// OUTPUT: Hi
```
